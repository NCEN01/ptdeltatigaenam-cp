# 📝 Panduan Revisi Konten Website — PT Delta Tiga Enam

Dokumen ini berisi **semua tulisan (judul, subjudul, deskripsi) yang tertanam langsung di dalam kode** website — dari halaman paling atas sampai bawah. Silakan bapak/ibu klien membacanya dan mengusulkan perbaikan kata-katanya.

## Cara memakai dokumen ini

1. Baca kolom **"Teks Sekarang"** — itulah tulisan yang tampil di website saat ini.
2. Tulis usulan perbaikan di kolom **"Ganti Menjadi"**. Bila sudah bagus/tidak perlu diubah, biarkan kosong.
3. **Bahasa Inggris tidak perlu diisi.** Cukup revisi versi Bahasa Indonesia — tim developer yang menyesuaikan versi Inggrisnya.
4. Kirimkan kembali dokumen yang sudah diisi; developer akan memasukkannya ke website.

> **Tidak ada di dokumen ini (karena bisa diubah sendiri lewat Panel Admin):** daftar Layanan, Portofolio, Artikel Blog, Agenda, Testimoni, Paket Kemitraan, data Sertifikat, serta gambar/banner.
>
> Beberapa teks bertanda **(default)** memiliki nilai bawaan di kode TAPI juga bisa ditimpa lewat Panel Admin (Settings). Bila diisi di admin, teks default tidak dipakai.
